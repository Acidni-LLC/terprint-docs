AI Recommender API

Integrate AI-powered strain recommendations into your cannabis applications using our REST API.

Introduction

The Terprint AI Recommender API provides intelligent cannabis strain recommendations based on:

  • Terpene Profiles: Match products by terpene composition
  • Desired Effects: Relaxation, focus, creativity, pain relief
  • Medical Conditions: Anxiety, insomnia, inflammation, etc.
  • User Preferences: THC tolerance, preferred formats, budget
  • Real-time Availability: Filter by what's in stock

Base URL: https://apim-terprint-dev.azure-api.net/recommend

Authentication

All API requests require authentication using an API key. Include your key in the request header:

curl -X GET "https://apim-terprint-dev.azure-api.net/recommend/api/recommendations" \
  -H "Ocp-Apim-Subscription-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json"

Get your API key: Sign up at the Developer Portal to receive your subscription key.

Get Recommendations

POST /api/recommendations

Get AI-powered strain recommendations based on user preferences and conditions.

{
  "effects": ["relaxation", "pain_relief"],
  "conditions": ["anxiety", "insomnia"],

Python SDK

Installation

pip install terprint-sdk

Usage

from terprint import TerprintClient

client = TerprintClient(api_key="your_api_key")

# Get recommendations
recommendations = client.recommend(
    effects=["relaxation", "pain_relief"],
    thc_range=(15, 25),
    limit=10
)

for rec in recommendations:
    print(f"{rec.strain_name} - {rec.match_score:.0%} match")
    print(f"  THC: {rec.thc}% | Price: ${rec.price}")

JavaScript SDK

Installation

npm install @terprint/sdk

Usage

import { TerprintClient } from '@terprint/sdk';

const client = new TerprintClient({ apiKey: 'your_api_key' });

const recommendations = await client.recommend({
  effects: ['relaxation', 'pain_relief'],
  thcRange: { min: 15, max: 25 },
  limit: 10
});

recommendations.forEach(rec => {
  console.log(`${rec.strainName} - ${rec.matchScore}% match`);
});

.NET SDK

Installation

dotnet add package Terprint.SDK

Usage

using Terprint.SDK;

var client = new TerprintClient("your_api_key");

var recommendations = await client.RecommendAsync(new RecommendRequest
{
    Effects = new[] { "relaxation", "pain_relief" },
    ThcRange = new Range(15, 25),
    Limit = 10
});

foreach (var rec in recommendations)
{
    Console.WriteLine($"{rec.StrainName} - {rec.MatchScore:P0} match");
}
